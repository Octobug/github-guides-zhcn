# 简介

> _译自 [Getting Started with GitHub Pages](https://guides.github.com/features/pages/)_

GitHub Pages 是通过 GitHub 发布的公共网页托管服务。最方便的方式是使用 Jekyll Theme Chooser 来加载已有的主题。之后，你可以通过 Web 或本地计算机远程修改 GitHub Pages 的内容和样式。
